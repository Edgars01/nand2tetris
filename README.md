# nand2tetris

https://www.nand2tetris.org/


Project 01

    Build the following gates:
    
    Elementary logic gates
    - Not -- DONE
    - And -- DONE
    - Or -- DONE
    - Xor
    - Mux
    - Dmux

    16-bit variants
    - Not16
    - And16
    - Or16
    - Mux16
    
    Multi-way variants
    - Or8Way
    - Mux4Way16
    - Mux8Way16
    - DMux4Way
    - DMux8Way
